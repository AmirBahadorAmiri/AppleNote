# AppleNote

پروژه **AppleNote** یه اپلیکیشن ساده Android-based با تم iOS (آپل) هست.  
دقیقاً برای شبیه‌سازی اپ Notes اپل طراحی شده (قالب، آیکون‌ها، فونت و رابط کاربری).
<br><br>
## ویژگی‌ها

- رابط کاربری کاملاً شبیه اپ Notes اپل (Dark/Light mode، آیکون‌های iOS و ...)
- ساخت با Kotlin + Android Studio
- پشتیبانی از Gradle (KTS)
- طراحی اولیه با Adobe XD (فایل .xd و اسکرین‌شات‌های دقیق)
<br><br>

## چطور شروع کنیم؟

کلون کنید:
   ```bash
   git clone https://github.com/AmirBahadorAmiri/AppleNote.git
   ```
